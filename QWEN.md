# Birthday Bot Project

Этот проект представляет собой бота для поздравления сотрудников с днем рождения в Telegram.

## Основные изменения от 07.04.2026

1.  **Проксирование запросов**: В связи с блокировками Telegram API на сервере `lan-install`, в бот добавлена поддержка SOCKS5-прокси (`127.0.0.1:1080`).
2.  **Архитектура с SSH-туннелем**: Бот использует SSH-туннель до `vpn-server` (Нидерланды) для обхода блокировок.
3.  **Отказоустойчивость**: Внедрен скрипт `run_bot.sh`, который автоматически проверяет работоспособность прокси перед запуском бота и восстанавливает туннель при необходимости.

## Состав проекта

*   `birthday_bot.c`: Исходный код бота на языке C (использует libpq и libcurl).
*   `run_bot.sh`: Скрипт-обертка для автоматической проверки связи и запуска бота.
*   `deploy.sh`: Скрипт для развертывания проекта на сервере.
*   `Makefile`: Описание процесса сборки бинарного файла.
*   `README_PROXY.md`: Подробная техническая документация по работе через прокси.

## Управление

*   **Локальная сборка**: `make`
*   **Деплой**: `./deploy.sh`
*   **Логи**: `~/birthday_bot/birthday_bot.log` на сервере.
