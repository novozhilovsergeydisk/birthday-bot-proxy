# Birthday Bot Proxy

Бот на языке C для поздравления сотрудников с днем рождения в Telegram, работающий через SOCKS5-прокси для обхода блокировок.

## Создание репозитория на GitHub

Для создания этого удаленного репозитория использовалась следующая команда GitHub CLI:

```bash
gh repo create birthday-bot-proxy --public --source=. --remote=origin --push
```

## Основные возможности
- **SOCKS5 Proxy**: Поддержка `socks5h://127.0.0.1:1080` для безопасного обхода сетевых ограничений.
- **Auto-repair**: Скрипт-обертка `run_bot.sh` автоматически восстанавливает SSH-туннель до `vpn-server` перед каждым запуском.
- **PostgreSQL**: Интеграция с базой данных сотрудников.
- **MarkdownV2**: Красивое форматирование сообщений с экранированием спецсимволов.

Подробная техническая документация по прокси находится в файле [README_PROXY.md](./README_PROXY.md).
